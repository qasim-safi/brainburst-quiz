## ✨ Features

- 🌍 **3 Categories** — General Knowledge, Technology, Science
- ⚡ **All Topics Mode** — 30 questions from all categories shuffled together
- ⏱️ **30-Second Timer** — Animated SVG countdown ring per question; turns red at 8s
- 🎯 **Time-Bonus Scoring** — Answer faster, score more points (up to 1,000 pts/question)
- 🔀 **Shuffled Questions & Answers** — Fresh every round, no memorizing positions
- ✅❌ **Instant Feedback** — Green flash for correct, red shake for wrong
- 🏆 **Results Screen** — Score, accuracy %, badge, emoji trophy
- 🎉 **Confetti Explosion** — Fires when accuracy ≥ 70%
- 💾 **High Scores** — Best score per category saved to `localStorage`
- ✨ **Particle Background** — Animated floating dots on home screen
- 📱 **Fully Responsive** — Works great on phones and desktops
